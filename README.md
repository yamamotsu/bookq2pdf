# bookq2pdf

BookQの内容をPDFに変換するChrome拡張機能

## Usage

拡張機能を有効にするとBookQ表示画面の右上に `DownloadPDF` が表示される．
クリックするとPDFの生成とダウンロードを開始します．

Click `Download PDF` to download pdf of the textbook.

![click `Download PDF`](images/bookq2pdf.jpg)
